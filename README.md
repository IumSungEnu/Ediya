# Ediya
Edita 키오스크 프로젝트

## 사용언어
- Java / HTML / CSS
## 개발환경
- eclipse

## 프로젝트 목표
1. 구매창(구매, 관리자)
2. 회원가입, 로그인
3. 구매 페이지
4. 재고 수량 페이지
5. 매출 관리
